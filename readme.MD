This repository is based on https://chatengine.io/
Used: ChatEngine, NodeJs, Vite, ReactJs

### How to run

1. Clone the repository
2. cd backend
3. Run `npm install`
4. Add privatekey in index.js
5. Run `npm run start`
6. cd ..
7. cd frontend
8. Run `npm install`
9. add projectid in .env.local  
```VITE_CHAT_ENGINE_PROJECT_ID = XXXXX ```
10. Run `npm run dev`
